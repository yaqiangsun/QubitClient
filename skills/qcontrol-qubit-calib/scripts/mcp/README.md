# 量子比特配置文件
qubit_config.json 是本项目的核心配置文件，用于统一管理 qr1、qr2、qr3 三个量子比特的硬件参数、实验配置与运行阈值，为各类量子测控、脉冲操控、频谱读取等实验提供基础参数支撑。

文件主体以量子比特名称作为一级索引，每个比特对象均标记类型为 qubit

内部涵盖多类精细化配置项：包含比特能级频率、谐振腔频率等频域参数；微波功率、相位、各类 π 脉冲幅度、时长、校正系数等脉冲控制参数；ADC 采样延迟、时序偏移、读取时长、读取功率等时序与读取配置；

同时记录了量子不同态对应的 IQ 坐标、非线性系数、Z 轴偏移等校准参数；

所有参数均标注对应物理单位，保证参数语义清晰。


# 设备配置文件

本配置文件用于统一管理量子测控系统内硬件设备信息与量子比特线路映射关系，是整个测控系统硬件组网、信号路由的核心依据，为脉冲输出、信号采集、设备通信提供底层配置支撑。

文件分为 device 和 wiring 两大模块，各司其职。

device 模块集中登记所有物理硬件设备信息，包含数模转换器 DAC、模数转换器 ADC、触发模块 TRIG、本振源 LO 等关键器件。每条设备配置记录了通信地址、采样率、启动延迟、硬件类型、设备型号、唯一编号、通信方式与固件版本等核心参数，明确了各设备的网络地址、工作参数及硬件属性，保障上位机可正常寻址、驱动硬件运行。

wiring 模块定义量子比特与硬件通道的物理接线逻辑，以量子比特为索引，区分 Z 轴调控、XY 平面操控、读取通道三大信号链路。配置中明确每个信号链路对应的 DAC/ADC 设备、物理通道编号及配套本振源，精准划定信号的输出、采集与射频信号来源，实现量子比特操控信号与硬件通道的一一绑定。
