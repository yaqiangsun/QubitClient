# QControl Qubit Calibration Skill

## Overview

QControl-qubit-calib is a measurement skill for the QControl quantum control system. It provides MCP tools for real-time measurement control and analysis pipeline for qubit calibration experiments.

## Features

- S21 spectroscopy and peak detection
- Rabi oscillation, Ramsey interferometry
- T1/T2 relaxation measurements
- Spectrum analysis (1D/2D)
- S21 vs Flux, Power shift
- Single-shot readout, DRAG optimization
- Optimal π-pulse calibration
- Randomized Benchmarking (1Q)
- Delta experiment

## MCP Integration

Provides MCP tools for orchestrating measurement workflows with real-time parameter updates and automated analysis.
