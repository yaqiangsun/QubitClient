# Ctrl.T1_2D 任务接口文档

## 概述

Ctrl.T1_2D 是 Ctrl 中的一个任务，用于执行二维T1测量，同时扫描Z脉冲幅度和延迟时间，获得T1随Z脉冲变化的二维图谱。

## 接口使用方式

### 客户端初始化

```python
from qubitclient.ctrl import QubitCtrlClient, CtrlTaskName

client = QubitCtrlClient()
```

### 请求参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| task_type | CtrlTaskName | 是 | 任务类型，固定为`CtrlTaskName.T1_2D` |
| qubits | list[str] | 是 | 要测量的量子比特列表，例如["Q0", "Q1"] |
| zpa_start | float | 否 | Z脉冲幅度起始点，默认值-1.0 |
| zpa_end | float | 否 | Z脉冲幅度结束点，默认值0.4 |
| zpa_sample_num | int | 否 | Z脉冲幅度采样点数，默认值71 |
| delay_start | float | 否 | 延迟时间起始点（秒），默认值0 |
| delay_end | float | 否 | 延迟时间结束点（秒），默认值80000 |
| delay_sample_num | int | 否 | 延迟时间采样点数，默认值17 |

### 调用示例

```python
# 执行二维T1测量
result = client.run(
    task_type=CtrlTaskName.T1_2D,
    qubits=["Q0"],
    zpa_start=-1.0,
    zpa_end=0.4,
    zpa_sample_num=71,
    delay_start=0,
    delay_end=80000,
    delay_sample_num=17
)

print(result)
```

## 返回值格式

返回的结果包含二维T1测量数据：

```json
{
  "data": {
    "Q0": {
      "zpa": [float, ...],
      "delay_time": [float, ...],
      "population": [[float, ...], ...]
    }
  },
  "parameters": {
    "zpa_start": -1.0,
    "zpa_end": 0.4,
    "zpa_sample_num": 71,
    "delay_start": 0,
    "delay_end": 80000,
    "delay_sample_num": 17
  }
}
```

### 字段说明

| 字段名 | 类型 | 描述 |
|--------|------|------|
| data | dict | 包含每个量子比特的二维T1测量数据 |
| zpa | List[float] | Z脉冲幅度扫描点 |
| delay_time | List[float] | 延迟时间扫描点（秒） |
| population | List[List[float]] | 二维布居数矩阵 |
| parameters | dict | 实验参数信息 |

## 应用场景

T1_2D测量主要用于：
- 研究T1随Z脉冲（偏置）变化的关系
- 寻找T1最优的工作点
- 分析量子比特能级结构对弛豫时间的影响
- 优化量子比特操作参数
