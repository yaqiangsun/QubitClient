
# 在QuLab中使用
## 配置
1. 将analysis目录复制到QuLab根目录下；
2. 将`config.py.example`拷贝为`config.py`,并配置相关参数

## 分析函数
```python
#####################################################################
# AI analysis
#####################################################################
from analysis.inception import s21
results = s21(image)
report.analysis = results
#####################################################################
```
将上述代码放在`template`对应的`analyze`函数结尾.

## 绘图函数
```python
#####################################################################
# 绘制analysis结果
#####################################################################
from analysis.visualization import plot_s21
plot_s21(report,basepath,name)
#####################################################################
```
将上述代码放在`template`对应的`plot`函数结尾.
