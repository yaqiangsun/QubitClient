# Ctrl.T1 任务接口文档

## 概述

Ctrl.T1 是 Ctrl 中的一个任务，用于执行T1弛豫时间测量，测量量子比特的能量弛豫时间，这是量子比特相干性的重要指标。

## 接口使用方式

### 客户端初始化

```python
from qubitclient.ctrl import QubitCtrlClient, CtrlTaskName

client = QubitCtrlClient()
```

### 请求参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| task_type | CtrlTaskName | 是 | 任务类型，固定为`CtrlTaskName.T1` |
| qubits | list[str] | 是 | 要测量的量子比特列表，例如["Q0", "Q1"] |
| delay | list[float] | 是 | 延迟时间扫描范围，例如[0, 1e-6, 2e-6, ..., 100e-6] (秒) |

### 调用示例

```python
# 执行T1弛豫时间测量
result = client.run(
    task_type=CtrlTaskName.T1,
    qubits=["Q0"],
    delay=[0, 1e-6, 2e-6, 5e-6, 10e-6, 20e-6, 50e-6, 100e-6]
)

print(result)
```

## 返回值格式

返回的结果包含T1弛豫数据和拟合参数：

```json
{
  "data": {
    "Q0": {
      "delay_time": [float, ...],
      "population": [float, ...],
      "t1_time": float,
      "decay_rate": float
    }
  },
  "parameters": {
    "delay_range": [0, 1e-4]
  }
}
```

### 字段说明

| 字段名 | 类型 | 描述 |
|--------|------|------|
| data | dict | 包含每个量子比特的T1测量数据 |
| delay_time | List[float] | 延迟时间扫描点（秒） |
| population | List[float] | 对应延迟时间下的激发态布居数 |
| t1_time | float | 能量弛豫时间T1（秒） |
| decay_rate | float | 衰减率（1/秒） |
| parameters | dict | 实验参数信息 |

## 应用场景

T1弛豫时间测量主要用于：
- 表征量子比特的能量弛豫特性
- 评估量子比特的相干性能
- 优化量子门操作的时间参数
- 研究环境噪声对量子比特的影响
